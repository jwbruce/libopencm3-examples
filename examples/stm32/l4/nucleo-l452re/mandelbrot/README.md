# README

This example program demonstrates the floating point instructions
on the ST Nucleo-L452RE (STM32L452RE) eval board.

A mandelbrot fractal is calculated and sent as "ascii-art"
image over USART2 at 115.2Kbaud/8N1.  UART sends are blocking.
To properly see the Mandelbrot "beetle" image, the serial output
will need to be captured to a file.

**LD2** toggle after each frame of the fractal is drawn 
to indicate that the program is running properly.  Frame rate is
not constant due to the varying number of iterations done in the
Mandelbrot calculations.


## Board connections

| Port  | Function      | Description                        |
| ----- | ------------- | ---------------------------------  |
| `PA2` | `(USART2_TX)` | TTL serial output `(115200,8,N,1)` |
